# 路线图

> 当前版本: ver 0.10-beta

本文档记录 CwtshTodo 项目的未来规划路线图，分为短期、中期和长期目标。

---

## [F01].[短期目标]

### [F01a].[完善高级功能]：（完成 M10 剩余任务）

#### [F01a01].[新增：任务标签系统] → [M10e] / [P03a]

目标内容：实现自定义标签系统，支持多标签关联和按标签筛选

规划：

- 创建 Tag 类型定义
- 实现标签的 CRUD 操作
- 任务与标签多对多关联
- 添加按标签筛选功能
- UI 界面支持标签显示和选择

#### [F01a02].[新增：重复任务生成] → [M10f] / [P03b]

目标内容：自动生成重复任务的子任务实例

规划：

- 设计重复任务实例生成逻辑
- 实现子任务完成状态独立追踪
- 添加重复规则可视化展示
- 处理重复任务的删除和修改

#### [F01a03].[新增：自定义番茄钟时长] → [M10g] / [P03c]

目标内容：允许用户自定义专注时长和休息时长

规划：

- 在设置中添加番茄钟配置选项
- 支持自定义专注时长（15-120 分钟）
- 支持自定义短休息和长休息时长
- 保存用户偏好设置到本地存储
- 专注模式页面应用自定义设置

### [F01b].[用户体验优化]：（提升用户交互体验）

#### [F01b01].[优化：邮件发送报告] → [M10h] / [P03d]

目标内容：实现邮件发送报告功能

规划：

- 设计邮件报告模板
- 实现 mailto 链接生成
- 添加 SMTP 集成选项（可选）
- 报告分享界面添加邮件发送按钮

#### [F01b02].[优化：国际化支持] → [M10i] / [P03e]

目标内容：实现中英文界面切换

规划：

- 提取所有 UI 文本到语言文件
- 实现 i18n 框架集成
- 添加语言切换设置
- 日期格式本地化处理

#### [F01b03].[优化：任务搜索高亮] → [M10j] / [P03f]

目标内容：实现搜索结果关键词高亮

规划：

- 搜索结果渲染时匹配文本高亮
- 支持正则表达式搜索（可选）
- 优化搜索性能

---

## [F02].[中期目标]

### [F02a].[数据增强]：（扩展数据管理能力）

#### [F02a01].[新增：任务导出图片] → [M11a] / [P02a]

目标内容：实现任务导出为图片功能

规划：

- 使用 Canvas 渲染任务内容
- 支持 PNG 格式导出
- 添加导出质量选项
- 任务卡片快速操作中添加导出按钮

#### [F02a02].[新增：云端同步] → [M11b] / [P02b]

目标内容：实现跨设备数据同步

规划：

- 设计数据同步架构
- 选择云存储方案（Firebase / 其他）
- 实现用户认证
- 数据加密传输
- 冲突解决策略

### [F02b].[测试保障]：（提升代码质量）

#### [F02b01].[新增：单元测试]

目标内容：为核心功能添加单元测试

规划：

- 使用 Vitest 测试框架
- 测试组件渲染
- 测试 store 状态管理
- 测试工具函数

#### [F02b02].[新增：类型完善]

目标内容：完善 TypeScript 类型注解

规划：

- 检查并补充缺失的类型定义
- 启用更严格的类型检查
- 消除 IDE 类型警告

---

## [F03].[长期目标]

### [F03a].[功能扩展]：（新增高级功能）

#### [F03a01].[新增：数据统计可视化]

目标内容：添加更多数据分析和可视化图表

规划：

- 任务完成趋势图
- 习惯达成率日历热图
- 时间分配饼图
- 专注时长统计

#### [F03a02].[新增：任务协作]

目标内容：支持多用户任务协作

规划：

- 团队任务共享
- 任务分配功能
- 协作评论
- 权限管理

#### [F03a03].[新增：智能建议]

目标内容：基于历史数据提供智能建议

规划：

- 任务耗时预测
- 最佳专注时段建议
- 习惯养成提醒
- 任务优先级建议

### [F03b].[平台扩展]：（扩展支持平台）

#### [F03b01].[新增：桌面应用]

目标内容：打包为桌面应用

规划：

- 使用 Tauri 或 Electron
- 原生系统集成
- 系统通知增强
- 快捷键全局支持

#### [F03b02].[新增：移动端 App]

目标内容：提供原生移动应用

规划：

- React Native 或 Flutter
- 推送通知
- 离线优先架构
- 小组件支持

### [F03c].[生态建设]：（建设开发者生态）

#### [F03c01].[新增：开放 API]

目标内容：提供公开 API 供第三方集成

规划：

- RESTful API 设计
- 认证机制
- 文档网站
- SDK 支持

#### [F03c02].[新增：插件系统]

目标内容：支持第三方插件扩展

规划：

- 插件 API 设计
- 插件市场
- 沙箱安全机制
